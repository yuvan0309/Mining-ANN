# Mining Factor of Safety (FoS) Prediction

Machine Learning prediction system for mining slope stability analysis.

## Features
- 5 ML models (SVM, Random Forest, XGBoost, LightGBM, ANN)
- Interactive gauge visualization
- Real-time predictions
- 94.98% accuracy (SVM model)

## Live Demo
Deployed on Vercel - serverless architecture

## Tech Stack
- Python 3.9
- scikit-learn, XGBoost, LightGBM
- Serverless deployment
